# Interactive Portfolio (HTML/CSS/JS Version)

This is a modern, interactive portfolio website built with vanilla HTML, CSS, and JavaScript, inspired by [Brittany Chiang's portfolio](https://brittanychinag.com).

## Persona

**Role:** Creative Developer

## Tech Stack

- **HTML5**
- **CSS3** (Custom Properties, Flexbox, Grid)
- **JavaScript** (ES6+)
- **Feather Icons** (via CDN)

## Getting Started

1.  Simply open `index.html` in your web browser.
    -   Or use a live server extension in VS Code.

## Customization

-   **Content:** Edit `index.html` to change text and links.
-   **Styles:** Edit `style.css` to change colors (CSS variables at the top) and layout.
-   **Logic:** Edit `script.js` for interaction behavior.
